商品名の自動取得(外部データベース)

楽天ウェブサービス(Rakuten Developers)の Application ID と Access Key を設定すると、スキャンしたJANコードから商品名を自動取得します。

データの控え(バックアップ・引き継ぎ)

在庫・商品マスタ・各種設定をファイル1つに保存できます。機種変更やアドレス変更の際は、変更前に「控えを保存」し、変更後の画面で「控えから復元」してください。
